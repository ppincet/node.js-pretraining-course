## Task 10: Create a ToDoManager Class

**Description:**  
Create a class `ToDoManager` with methods to add, remove, mark as completed, and get all todos. Use the previous functions as methods.

**Example:**
```js
const manager = new ToDoManager();
manager.add({ id: 1, title: 'Buy milk', completed: false });
manager.markCompleted(1);
manager.getAll(); // [{ id: 1, title: 'Buy milk', completed: true }]
```
