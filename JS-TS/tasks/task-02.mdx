## Task 2: Filter Incomplete ToDos

**Description:**  
Write a function `getIncompleteTodos` that takes an array of todos and returns only those where `completed` is false.

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false },
  { id: 2, title: 'Read book', completed: true }
];
getIncompleteTodos(todos); // [{ id: 1, title: 'Buy milk', completed: false }]
```
