## Task 1: Create a ToDo Array Structure

**Description:**  
Create an array of objects to represent ToDo items. Each item should have an `id`, `title`, and `completed` (boolean).

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false },
  { id: 2, title: 'Read book', completed: true }
];
```
