## Task 7: Get ToDo Titles

**Description:**  
Write a function `getTitles` that takes an array of todos and returns an array of their titles.

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false },
  { id: 2, title: 'Read book', completed: false }
];
getTitles(todos); // ['Buy milk', 'Read book']
```
