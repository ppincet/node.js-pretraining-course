## Task 6: Remove a ToDo by ID

**Description:**  
Write a function `removeTodo` that takes an array of todos and an id, and returns a new array without the todo with the given id.

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false },
  { id: 2, title: 'Read book', completed: false }
];
removeTodo(todos, 1); // [{ id: 2, title: 'Read book', completed: false }]
```
