## Task 4: Define a ToDo Type

**Description:**  
Define a TypeScript type `ToDo` with fields: `id` (number), `title` (string), and `completed` (boolean).

**Example:**
```ts
const todo: ToDo = { id: 1, title: 'Buy milk', completed: false };
```
