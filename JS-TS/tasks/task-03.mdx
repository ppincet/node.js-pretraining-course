## Task 3: Add a New ToDo

**Description:**  
Write a function `addTodo` that takes an array of todos and a new todo object, and returns a new array with the new todo added.

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false }
];
const newTodo = { id: 2, title: 'Read book', completed: false };
addTodo(todos, newTodo); // [ { id: 1, ... }, { id: 2, ... } ]
```
