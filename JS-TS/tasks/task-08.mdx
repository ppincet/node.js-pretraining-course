## Task 8: Create ToDo with Default Values

**Description:**  
Write a function `createTodo` that takes a title and returns a todo object with a unique id and `completed` defaulting to false.

**Example:**
```js
const todo = createTodo('Go jogging');
// { id: <number>, title: 'Go jogging', completed: false }
```
