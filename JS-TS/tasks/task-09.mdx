## Task 9: Get Unique ToDo Titles

**Description:**  
Write a function `uniqueTitles` that takes an array of todos and returns an array of unique titles.

**Example:**
```js
const todos = [
  { id: 1, title: 'Buy milk', completed: false },
  { id: 2, title: 'Buy milk', completed: true },
  { id: 3, title: 'Read book', completed: false }
];
uniqueTitles(todos); // ['Buy milk', 'Read book']
```
