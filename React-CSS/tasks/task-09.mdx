## Task 9: Wrap ToDoList in a Card Component

**Description:**  
Create a `Card` component that wraps its children in a styled div. Use it to wrap the ToDoList.

**Example:**
```jsx
<Card><ToDoList todos={...} /></Card>
// Output: <div class="card">...</div>
```
